About

This is a fan-made visualizer for the music of Deltarune. Songs are displayed on a graph and connected by arrows which signify leitmotifs and melodies. It's currently a work in progress.

Disclaimers

This is a project to compile community knowledge, so it's dependent on work that came before. Check out the Thanks section for useful links to resources that provide more detailed analysis.

Personal interpretation is needed to identify leitmotifs and connections between melodies. This is my own attempt, so do use your own judgement and compare resources before coming to conclusions.

FAQ

Q: Will the Undertale soundtrack be added?

For now this project is focused on Deltarune, and will include Undertale songs only when directly relevant.

Q: I noticed a mistake. / How can I contribute?

You can send me a message on Discord (@moxis) or Bluesky (coming soon). I'm happy to receive any comments or suggestions. Thank you!
